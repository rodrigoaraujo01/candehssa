# Candehssa

Isso é um aplicativo para um grupo de RPG

Os PJs estão sediados em uma cidade chamada Candeh'ssa
Nessa cidade, há 10 locais
Em cada um dos locais, há entre 1 e 4 NPCs
Cada PJ pode realizar interações com os NPCs e obter até 7 Pontos de Amizade 
com cada um dos NPCs
Diferentes níveis de Pontos de Amizade dão acesso a diferentes habilidades, 
poderes e benesses para os PJs
Ao atingir 7 Pontos de Amizade com um dado NPC, aquele jogador libera uma missão
especial com esse NPC
Essa missão libera a habilidade final do NPC

## Objetivo

O objetivo é construir um aplicativo html que possa ser disponibilizado através
de uma página estática no GitHub
Para armazenamento de dados, há um banco de dados Supabase (Candehssa https://ffgpaipeomjrimufmpff.supabase.co)
O usuário poderá cadastrar seu PJ (nome, raça e nível) e começar a rastrear 
os seus Pontos de Amizade com os NPCs
O usuário poderá cadastrar um grupo de aventureiros (nome). Outros usuários 
poderão entrar nesse grupo desde que saibam o nome e a senha do grupo
Os NPCs só se tornam disponíveis depois que algum membro visita seu local e o 
encontra
Uma vez que um membro do grupo conheça um NPC, ele se torna visível para todos
os outros
Para conhecer um NPC, a interface deve permitir clicar em um local e em um nome 
de NPC
